User Feedback durch Parametrisches Design

  1. Der/Die Designer:in definiert, über welche Aspekte vom Design sie Feedback einholen möchte
  2. Damit User:innen mit dem Produkt interagieren und Feedback abgeben können, muss ein parametrisches Modell des ganzen Produktes oder Teilen des Produktes erstellt werden. In dieser Arbeit wird hierfür die Applikation Grasshopper verwendet
  3. Um das parametrische Modell im Web nutzen zu können, muss dieses für das Web konvertiert und jeweils neu berechnet werden. In dieser Arbeit wird hierfür die Webseite Shapediver genutzt
  4. Auf der Webseite können Benutzer:innen Parameter im Rahmen des Feedbacks nach ihren Wünschen und Anforderungen anpassen
  5. Die eingegebenen Parameter werden in einer Datenbank gespeichert und können durch die Designenden ausgewertet werden
